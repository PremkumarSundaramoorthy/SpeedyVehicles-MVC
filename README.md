
# ASP.NET Core MVC application
It is a MVC web application with Entity Framework and SQL server.

## Technologies used

 - **Frontend** 
   - HTML 
   - Bootstrap 
   - Razor Views
 - **Backend** 
   - C#
   - ASP.NET Core MVC
   - Entity Framework Core
 - **Database**
   - SQL Server

## ASP.NET Core MVC Features Used

- N-Tier Architecture
- Repository pattern
- Unit of Work pattern
- Authentication
- Authorization
- Area 
- Identity
- Role Based Access
- Logging
